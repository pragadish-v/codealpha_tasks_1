# 🛍️ Nexora Platform - Full Stack E-Commerce Website

> A modern full-stack e-commerce platform built with the MERN stack featuring secure authentication, product management, shopping cart, and responsive UI.

🌐 **Live Demo:** https://codealpha-tasks-1-k0u9.onrender.com/

---

## 📌 About

Nexora is a complete e-commerce application developed as **Task 1** for the **CodeAlpha Web Development Internship**.

The project focuses on creating a real-world online shopping platform with authentication, product browsing, cart management, and an admin dashboard while following clean architecture and responsive design principles.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/nexora-platform.git

cd nexora-platform
```

## Install Dependencies

```bash
npm install
```

## Create Environment Variables

Create a `.env` file.

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

JWT_EXPIRES_IN=7d
```

## Run Development Server

```bash
npm run dev
```

or

```bash
npm start
```

Open:

```
http://localhost:5000
```
